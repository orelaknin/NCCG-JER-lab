Zabbix API Automation Scripts
This repository contains a collection of Python scripts designed to interact with the Zabbix API. 
All scripts are organized within the zabbix_api folder and are further divided into three subfolders: backup, create, and update.

zabbix_api/backup/: Contains scripts to back up Zabbix inventory, groups, tags, and more.
zabbix_api/create/: Contains scripts to create SNMP hosts, delete hosts, and other creation-related tasks.
zabbix_api/update/: Contains scripts to update host inventory, tags, VLAN groups, and more.
